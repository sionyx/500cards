# Генератор карточек для игры аналогичной игре "500 злобных карт"

Этот скрипт создает PDF-файл с карточками (3×6 карточек на страницу).

## Установка

### 1. Создание виртуального окружения

Для создания виртуального окружения выполните команду:

```bash
python -m venv venv
```

### 2. Активация виртуального окружения

На Windows:
```bash
venv\Scripts\activate
```

На macOS и Linux:
```bash
source venv/bin/activate
```

### 3. Установка зависимостей

После активации виртуального окружения установите зависимости:

```bash
pip install -r requirements.txt
```

## Использование

Для запуска скрипта используйте команду:

```bash
python script.py -i 100redcards.txt -o 100redcards.pdf
python script.py -i 300whitecards.txt -o 300whitecards.pdf
```

Где:
- `-i 100redcards.txt` - путь к файлу с фразами (по одной на строку)
- `-o 100redcards.pdf` - путь к выходному PDF-файлу
